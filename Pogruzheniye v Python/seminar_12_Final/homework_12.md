## Урок 12. ООП. Финал

### Задачи:

Задание №1

Создайте класс студента.

1. Используя дескрипторы проверяйте ФИО на первую заглавную букву и
   наличие только букв.
2. Названия предметов должны загружаться из файла CSV при создании экземпляра.
   Другие предметы в экземпляре недопустимы.
3. Для каждого предмета можно хранить оценки (от 2 до 5) и результаты
   тестов (от 0 до 100).
4. Также экземпляр должен сообщать средний балл по тестам для каждого
   предмета и по оценкам всех предметов вместе взятых.
